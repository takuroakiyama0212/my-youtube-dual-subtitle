How to Install

Since this is a developer version, installation takes 30 seconds.

1

Download the files

Click the download button above to get the Dual Subtitles.zip file. Unzip it into a folder on your computer.

2

Open Chrome Extensions

Type chrome://extensions in your address bar and press Enter.

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top right corner of the extensions page.

4

Load Unpacked

Click the "Load unpacked" button and select the folder where you unzipped the files.
